# Next.js Page Components

A collection of reusable Next.js page components for PWA applications.

## Installation
```bash
npm install @your-username/nextjs-page-components
```

## Components

- **LoginPage** - OTP-based login page with mobile/web responsive views
- **DashboardPage** - Dashboard layout with sidebar and header
- **ProfilePage** - User profile page
- **SettingsPage** - Settings page

## Usage

### LoginPage
```javascript
import { LoginPage } from "@your-username/nextjs-page-components";
// See LoginPage/README.md for detailed usage
```

### DashboardPage
```javascript
import { DashboardPage } from "@your-username/nextjs-page-components";
// See DashboardPage/README.md for detailed usage
```
